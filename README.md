# Calculadora De Calorias
Proyecto personal para calcular calorias
## OBJETIVO
Permitir al usuario registrar el alimento y calcular
- Calorias
- Proteina
- Carbohidratos
- Azucares
- Grasas

## FUNCIONALIDADES
### AÑADIR ALIMENTOS
El usuarario podra: 
- Buscar alimentos
- Introducir Gramos
- Seleccior crudo o cocinado

### CALCULO AUTOMATICO
La aplicacion calculara automaticamente segun los gramos introducidos
- Calorias (kcal)
- Proteina
- Carbohidratos
- Azucares
- Grasas

### GUARDADO
Los alimentos se guardar automaticamente en la etapa de la comida seleccionada:
- Desayuno
- Comida
- Merienda
- Cena
- Post-cena
